LinkedHashSet

LinkedHashSet是不含重复元素的集合、最多包含一个null、元素是有序的、线程不安全（效率高）

LinkedHashSet与HashSet对比就多了一个顺序，应用的不多

TreeSet

TreeSet可以支持自定义排序。如果TreeSet所储存的对象的类没有实现Comparable接口就会报错ClassCastException。 所以要使用TreeSet来对自定义对象排序必须实现Comparable接口

特点：元素唯一性、可以自定义排序、不允许null存在、不是线程安全

1.排序比较

*字符串的比较规则是先按第一个字符来比较，之后按第二个、第三个.......

2.类如果要实现比较的规则都会要实现Comparable接口
