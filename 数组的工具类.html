数组的工具类Arrays

方法摘要

static String toString()
返回指定数组内容的字符串表示形式。
static void sort
对数组进行排序。
static int binarySearch([],key)
二分搜索法,来查找key的索引 		适合对有序的数组进行查找
static < T > List< T > asList(T... a)
返回一个受指定数组支持的固定大小的列表。

我们不能把基本数据类型的数组转换成集合（int为基本数据类型，Integer不为基本数据类型，用Integer代替int使用asList方法)
如果传了，就会把数组的类型作为泛型
