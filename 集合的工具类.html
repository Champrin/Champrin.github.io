集合的工具类Collections

Collection与Collections的区别为：前者是集合的接口，后者是操作集合的工具类

方法摘要

static void reverse(List list)
反转指定列表中元素的顺序。		 集合反转
static void shuffle(List list)
使用默认随机源对指定列表进行置换。		 打乱集合
static < T extends Comparable < ? super T> > void sort(List< T > list)
根据元素的自然顺序 对指定列表按升序进行排序。 		使用要求：此列表中的对象的类要实现Comparable接口
static < T > void sort(List< T > list, Comparator c)
根据指定比较器产生的顺序对指定列表进行排序。		 用外部比较器来对列表进行排序
synchronized(List,Set,Map)
返回一个线程安全的这个列表
static < T > List< T > asList(T... a)
返回一个受指定数组支持的固定大小的列表。

我们不能把基本数据类型的数组转换成集合（int为基本数据类型，Integer不为基本数据类型，用Integer代替int使用asList方法)
如果传了，就会把数组的类型作为泛型
