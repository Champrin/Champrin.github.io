Java-List

Java List 学习记录

List是有序的集合，像数组一样。可以把List理解为是一个长度可变的数组，而且有丰富的api。List集合底层就是数组

List的方法继承了Collection，操作List可回顾Java集合 学习记录

List的元素可以重复

void add(int index, E element)
在列表的指定位置插入指定元素（可选操作）。 		在index处添加item元素

在一处插入一个元素，后面的元素都会向后移动一位

E get(int index)
返回列表中指定位置的元素		 根据索引号来获得这个索引的内容
E set(int index, E element)
用指定元素替换列表中指定位置的元素（可选操作）。		 将一个索引号的值改为element
int indexOf(Object o)
返回此列表中第一次出现的指定元素的索引；如果此列表不包含该元素，则返回 -1。
int lastIndexOf(Object o)
返回此列表中最后出现的指定元素的索引；如果列表不包含此元素，则返回 -1。

List的迭代器

      List的迭代器 遍历过程中也是不允许对集合添加删除的变更操作

ListIterator < E > listIterator()
返回此列表元素的列表迭代器（按适当顺序）。 		获得这个List集合的迭代器
boolean hasPrevious()
如果以逆向遍历列表，列表迭代器有多个元素，则返回 true。 		判断迭代器中是否有前一个元素
E E previous()
返回列表中的前一个元素。

      逆向迭代 


        List collection = new ArrayList();
        collection.add("value");
        ListIterator iterator = collection.listIterator();
        while(iterator.hasPrevious()){
            String obj = (String) iterator.previous();
            System.out.println(obj);
        }

      这里提供了一个可以进行添删操作的方法，但是遍历的本次不生效 


        List collection = new ArrayList();
        collection.add("value");
        ListIterator iterator = collection.listIterator();
        while(iterator.hasNext()){
            String obj = (String) iterator.next();
            iterator.add("ok"); //在这里
            System.out.println(obj);

        }



        输出结果仍为value，而不是value ok

      若要实时添加删除的变更操作，可以用for 


        List collection = new ArrayList();
        collection.add("value");
        for (int i = 0; i < collection.size(); i++) {
            String obj = (String) collection.get(i);
            if ("value".equals(obj)){
                collection.add("water");
            }
            System.out.println(obj);
        }



        输出结果为value water

ArrayList

      特点：元素可以重复、可以有null存在、有顺序，长度不固定、线程不安全、效率高

      在添加数据的时候初始长度是10，后续以5个长度来扩充集合的长度

      好处：性能高 缺点：线程不安全，可能带来数据不准确

LinkedList

      堆栈：先进后出 队列：先进先出

      链表：储存了元素和其前后地址，使元素之间互相联系起来

       LinkedList是以链表为原理，添加修改删除的性能高，不是线程安全的

void addFirst(E e)
将指定元素插入此列表的开头。
void void addLast(E e)
将指定元素添加到此列表的结尾。
E removeFirst()
移除并返回此列表的第一个元素。
E removeLast()
移除并返回此列表的最后一个元素。
E pop()
从此列表所表示的堆栈处弹出一个元素。
void push(E e)
将元素推入此列表所表示的堆栈。
